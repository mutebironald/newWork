# Demo Script

## Opening
NewWork is a work-creation operating system. It helps under-employed youth become AI-assisted field agents for informal merchants, then proves the outcome through payments, proof, and partner dashboards.

## Demo flow
1. A field agent signs up.
2. The agent completes a skill scan.
3. NewWork's AI Work Designer generates a skill profile.
4. AI recommends merchant-service offers.
5. The agent selects WhatsApp Catalog Support.
6. The agent adds a merchant.
7. AI generates outreach and a delivery checklist.
8. The agent logs a paid work episode.
9. The agent uploads proof.
10. The admin dashboard updates Business Viability, AI-Native Operations, and Category Impact metrics.

## Closing
NewWork is not a job board and not a generic chatbot. It creates a measurable path from hidden skills to paid work, and it gives organizations evidence that income was created.
